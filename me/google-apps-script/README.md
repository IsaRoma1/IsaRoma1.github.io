# Roman ME API

Готовый Google Apps Script для связи `https://ii41.ru/me/` с таблицей
`Roman ME — база личной системы`.

## Установка

1. Открыть таблицу.
2. Выбрать `Расширения → Apps Script`.
3. Удалить пример `myFunction` из `Code.gs`.
4. Вставить содержимое файла `Code.gs` и сохранить.
5. Запустить функцию `testSetup` и выдать разрешения Google.
6. Выбрать `Развернуть → Новое развертывание → Веб-приложение`.
7. Выполнять от имени владельца. Доступ: `Все`.
8. Скопировать URL развертывания, оканчивающийся на `/exec`.

URL нужно передать Северу: он подключит его к сайту и проведёт сквозной тест записи.
